Convert Code to MD

Generates Markdown files from a source code folder. Each output file represents one source file and lists the local code dependencies it imports or references, so the generated folder can be opened as a dependency map in the graph view.

Supported languages: JavaScript, TypeScript, Python, Java, and C#. Supported extensions: .js, .jsx, .mjs, .cjs, .ts, .tsx, .py, .java, and .cs.
